# Sales Data Analysis using SQL

## Overview
This project analyzes sales data using SQL to extract business insights on customer behavior, product performance, and revenue trends.

## Tools Used
- MySQL
- SQL (Joins, Aggregations, Window Functions)

## Dataset
- Customers
- Products
- Orders

## Key Analysis
- Total Sales & Profit
- Top Customers
- Sales by Category
- Monthly Sales Trends
- Customer Ranking

## Project Structure
- project.sql → full SQL script
- dataset → raw data
- output → query results

## Key Learnings
- Data modeling
- Writing complex SQL queries
- Business insight generation
